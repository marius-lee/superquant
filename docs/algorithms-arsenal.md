# 算法武库 — 攻击性涨停预测 (2026-06-24)

## 第一批：已验证的经典算法

### 1. XGBoost/LightGBM 涨停分类
**数据**: 5分钟线 + 资金流 + 龙虎榜 (全部免费)
**标签**: 当日是否涨停 (binary)
**优势**: 非线性特征交互, 自动发现复杂模式
**优先级**: 🔴 立即

### 2. 贝叶斯变点检测
**数据**: 分钟级价格/成交量序列
**原理**: P(变点|数据) ∝ P(数据|变点) × P(变点)
**优势**: 涨停前必有突变 — 量/价/量价同时突变
**优先级**: 🟡

### 3. 霍克斯过程 (订单流)
**数据**: easy-tdx 逐笔买卖
**原理**: λ(t) = μ + α × Σ(历史买入衰减)
**优势**: 买入自我激发 → 封板前兆
**优先级**: 🟡

### 4. 极端值理论 (GEV)
**数据**: 历史涨停样本
**原理**: GEV 尾部比正态厚 20 倍
**优势**: 正确评估极端事件概率
**优先级**: 🔵

### 5. 汤普森采样 (策略选择)
**数据**: 各策略历史盈亏
**原理**: Beta 分布采样 → 自适应探索/利用
**优势**: 自动发现"今天最有效的策略"
**优先级**: 🟡

### 6. 保形预测
**数据**: 历史预测+实际结果
**原理**: 有数学保证的置信区间 (既定覆盖水平)
**优势**: 不是点估计, 是可信区间
**优先级**: 🔵

## 第二批：深度思考产出

### 7. 生存分析 (Cox比例风险)
**问题**: 不是"会不会涨停", 而是"多久会涨停"
**数据**: 分钟级价格 + 封板时间标签
**原理**: h(t|X) = h₀(t) × exp(β₁X₁ + ...)
**优势**: 对"今天不会涨停"的股票, 模型自动给低分
**实现**: `from lifelines import CoxPHFitter`
**优先级**: 🔴 立即可做 (纯Python包)

### 8. 孤立森林异常检测
**问题**: 正常股票太多了, 只关注异常的
**数据**: 分钟级 OHLCV
**原理**: 随机森林隔离异常点, 异常=涨停前兆
**优势**: 无监督, 不需要标签, 自动发现新奇模式
**实现**: `from sklearn.ensemble import IsolationForest`
**优先级**: 🔴 立即可做

### 9. 隐马尔可夫模型 (市场状态)
**问题**: 牛市/熊市/震荡市 → 不同策略最有效
**数据**: 大盘指数分钟线
**原理**: 隐藏状态转移 + 观测状态发射
**优势**: 自动识别"现在是哪个市场状态"
**实现**: `from hmmlearn import GaussianHMM`
**优先级**: 🟡 本周

### 10. 卡尔曼滤波器 (价格动量)
**问题**: 量变化快, 价格变化慢, 需要融合
**数据**: 分钟级价格+成交额
**原理**: 预测-更新循环: x̂_t = Fx̂_{t-1} + K_t(y_t - Hx̂_{t-1})
**优势**: 实时平滑估计"真实价格趋势"
**优先级**: 🟡 本周

### 11. 图神经网络 (板块关联)
**问题**: A股的股票通过板块相互关联
**数据**: 股票归属概念板块 (AData已有)
**原理**: 股票=节点, 同板块=边 → GCN
**优势**: 一只股票的涨停概率受同板块其他股票影响
**实现**: `from dgl import DGLGraph` 或 `torch_geometric`
**优先级**: 🟢 下周

### 12. 元学习 MAML (跨市场适应)
**问题**: 模型在牛市中训练的, 到了熊市就失效
**数据**: 不同市场时期的数据
**原理**: 学习一个"能快速适应新环境"的初始参数
**优势**: 换市场环境不换模型, 自适应
**优先级**: 🔵 研究

### 13. 对抗验证 (分布偏移检测)
**问题**: 历史数据训练的模型, 今天可能已过时
**数据**: 训练集 vs 新样本
**原理**: 训练一个二分类器区分"历史"vs"今天"
**优势**: 如果 AUC>0.7 → 市场变了 → 模型失效预警
**实现**: `from sklearn.ensemble import RandomForestClassifier`
**优先级**: 🟡 本周

### 14. 神经ODE (连续时间价格模型)
**问题**: 离散K线丢失了价格连续变化的信息
**原理**: dh/dt = f(h, t, θ) → 神经网络参数化 ODE
**优势**: 连续时间建模, 不受K线频率限制
**实现**: `from torchdiffeq import odeint`
**优先级**: 🔵 研究

### 15. 变分自编码器 (VAE) 涨停模式
**问题**: 涨停前的K线形态多种多样, 需要更丰富的表达
**数据**: 涨停前60分钟的K线序列
**原理**: 编码器压缩→潜在空间→解码器重建
**优势**: 潜在空间中"近邻=相似涨停模式"
**实现**: `import torch.nn as nn`
**优先级**: 🟢 下周

### 16. 压缩感知 (稀疏信号恢复)
**问题**: 5分钟K线是稀疏的 (48根/天), 丢失了高频信息
**原理**: 用 ℓ₁ 正则化从少量观测中恢复高频信号
**优势**: 用免费5分钟线近似L2高频数据的效果
**优先级**: 🔵 研究

### 17. 信息几何 (费舍尔信息矩阵)
**问题**: 如何测量"市场的不确定性在快速变化"
**原理**: 计算概率分布参数的费舍尔矩阵→瑞曼度量
**优势**: 费舍尔信息突变 → 市场结构改变 → 大机会来了
**优先级**: 🔵 研究

### 18. 深度强化学习 (DQN/PPO) 最优执行
**问题**: 什么时候买, 买多少, 什么时候卖 → 序列决策
**数据**: 分钟级价格 + 模拟买卖
**原理**: Q(s,a) = r + γ×max Q(s',a') → 学习最优策略
**优势**: 自动发现复杂的买卖时机, 不是固定规则
**实现**: `import stable_baselines3`
**优先级**: 🟡 本周
