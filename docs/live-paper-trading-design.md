## 二、数据流

```
 9:25 盘前
   │
   ├── 1. 从 Sina 拉取昨日日线 → 更新 Hikyuu HDF5
   ├── 2. 计算全市场因子截面评分 (P1 factors)
   └── 3. 输出「今日因子选股池」Top N
   
 9:30 开盘
   │
   ├── 4. Sina WebSocket 实时行情 → 写入内存缓存
   ├── 5. 每5分钟: 扫描陈小群信号 (SG)
   │       ├── 命中信号 → 查因子评分 (MF)
   │       │   ├── 因子高分 → 生成买入委托
   │       │   └── 因子低分 → 跳过
   │       └── 无信号 → 继续监控
   ├── 6. 持仓监控:
   │       ├── 止损检查 (ST: 波动率自适应)
   │       ├── 止盈检查 (TP: MCVA动态)
   │       └── 触发 → 生成卖出委托
   └── 7. 模拟成交:
           └── 以当前价立即成交 (无滑点模拟)
               └── 记录到 sim_trades 表

 15:00 收盘
   │
   ├── 8. 存储今日5分钟线 → HDF5
   ├── 9. 生成复盘报告
   └── 10. 更新绩效统计
```

## 三、模拟成交设计

### 3.1 为什么不用真实券商

| 真实券商 | 模拟交易 |
|---------|---------|
| 需要开户、入金 | 零成本 |
| T+1 限制 | 完全模拟 T+1 |
| 最小 100 股 | 灵活设置 |
| 佣金印花税 | 可配置费率 |
| 不能 7×24 回测 | 任意时间回放 |

### 3.2 模拟规则

```
买入:
  - 以信号触发时的「当前价」成交 (市价单)
  - T+1: 当日买入次日才能卖出
  - 最小单位: 100 股
  - 费率: 万三佣金 + 千一印花税 (卖时收)
  - 涨跌停限制: 涨停价不买, 跌停价不卖

卖出:
  - 止损/止盈触发时以「当前价」成交
  - 开盘卖出: A1-A4 规则以开盘价成交
  - 盘中卖出: B 链规则以触发价成交

账户:
  - 初始资金: ¥5,000 (北极星起点)
  - 持仓上限: 3 只 (config 可调)
  - 每笔记录: symbol, side, price, shares, pnl, timestamp
```

### 3.3 与 Hikyuu TradeManager 的关系

```
Hikyuu TradeManager (tm)
├── tm.current_cash       → 可用资金
├── tm.get_position(sym)  → 持仓查询
├── tm.buy(sym, price, shares)   → 模拟买入
├── tm.sell(sym, price, shares)  → 模拟卖出
└── tm.get_funds_curve()  → 资金曲线

sim_trades 表 (results.db 或 trades.db)
├── date, time, symbol, side, price, shares
├── pnl, pnl_pct, capital_after
├── strategy, signal_source, factor_score
└── created_at
```

## 四、Flask Web 界面设计

### 4.1 现有 API (保留)

| 路由 | 用途 | 改动 |
|------|------|------|
| `/` | 主页面 | 新增因子面板 |
| `/api/state` | 实时状态 | 新增因子分数字段 |
| `/api/mood` | 情绪周期 | 不变 |
| `/api/signals` | 信号列表 | 新增因子置信列 |
| `/api/sectors` | 板块热度 | 不变 |
| `/api/positions` | 持仓 | 新增止损/止盈触发价 |
| `/api/trades` | 交易记录 | 新增策略标签 |
| `/api/review` | 盘后复盘 | 不变 |
| `/api/performance` | 绩效统计 | 新增 vs 北极星对比 |

### 4.2 新增 API

| 路由 | 用途 | 频率 |
|------|------|------|
| `/api/factors` | 今日因子截面 Top 20 + 因子分布 | 盘前更新 |
| `/api/factor-score?symbol=SH600000` | 单股票因子得分明细 | 按需 |
| `/api/paper-account` | 模拟账户摘要 (资金/持仓/盈亏) | 实时 (5s) |
| `/api/northstar` | 北极星进度 (¥5000→¥100万) | 实时 |
